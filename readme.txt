To get project runnung:
    1. Ensure you already have NPM installed (https://www.npmjs.com/get-npm)
    2. From CMD/ Terminal (or similar) navigate into root folder of this project
    3. Run command "npm install" to install all packages required to run this application
    4. Next, run command "npm start". This will start both the json-server
        and the react application.
